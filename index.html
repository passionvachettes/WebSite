La course Landaise

est un spectacle de vachettes, inoffensif pour les animaux.

La ganaderia AVENTURA organise deux types de spectacles en une ou deux parties :

+ Une course landaise intégrale avec écarts et sauts de vachettes landaises,

+ Une démonstration de course landaise suivie de jeux avec les vachettes pour le public. Une occasion pour les spectateurs de venir affronter les vaches lors des jeux proposés (foot, balançoire, anneaux, cocarde, petit veau pour les enfants).

Nous nous déplaçons chez-vous, dans toute la France. Le seul pré-requis est un terrain plat de xx m2 pour l'installation de l'arène et des gradins.
